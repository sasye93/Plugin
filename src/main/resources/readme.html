Containerization Plugin for ScalaLoci v1.0

Prerequisities

  • Scala 2.12.6+
  • A ScalaLoci project with sbt
  • A running installation of the Docker Deamon (Docker for Windows)
  • Shell bash support on Windows (e.g. cygwin64)

Folder structure

  • container /

    The created Dockerfile which is used to create Docker Images for this Peer (either automatically by the plugin or manually).

    • Dockerfile

      The created Dockerfile which is used to create Docker Images for this Peer (either automatically by the plugin or manually).

    • run.sh

      The created Dockerfile which is used to create Docker Images for this Peer (either automatically by the plugin or manually).

"Dockerfile\t\t: \n" + "run.{sh|cmd}\t\t: The run script for the Container, will be called as entry point when the container is started and start the .jar.\n" + "BuildContainer.{sh|cmd}\t: Script that can be used to build this Container (Build a Docker Image).\n" + "RunContainer.{sh|cmd}\t: Script that can be used to run this Container (first, an image must be build using the Dockerfile, either automatically or manually).\n" + "StopContainer.{sh|cmd}\t: Script that can be used to stop this Container.\n" + "*.jar\t\t\t: The actual tier code, packed as JAR.\n"
